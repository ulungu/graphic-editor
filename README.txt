coursework on object-oriented programming on the topic "Graphic editor". 
A system of classes for describing flat geometric figures has been implemented: line, circle, square, rectangle, as well as methods for creating objects, moving on a plane, changing sizes and rotating by 90 degrees. 
The program contains:
- interface classes;
- abstract class;
- a real class or a set of real classes that allow you to solve the problem;
- events and event handlers;
- emergency handlers
